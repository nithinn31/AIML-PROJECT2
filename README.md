# AIML-PROJECT2
Student Performance Prediction using Machine Learning
This project predicts student performance based on factors like study hours, attendance, and previous scores using machine learning algorithms.
##Technologies Used
python
Pandas
Scikit-learn
Matplotlib
##Student Performance Prediction using Machine Learning
Step 1: Import Libraries
Import required Python libraries such as Pandas, NumPy, Matplotlib, and Scikit-learn.
Step 2: Load Dataset
Load the dataset containing student details like study hours, attendance, and marks.
Step 3: Data Preprocessing
Handle missing values, encode categorical data, and scale the features.
Step 4: Apply Machine Learning Algorithm
Train models like Linear Regression, Decision Tree, and Random Forest.
Step 5: Visualization
Visualize data using graphs and compare actual vs predicted performance.
##Conclusion:
The machine learning models successfully predicted student performance based on input features.
Random Forest gave better accuracy compared to other models.
This helps in identifying students' strengths and improving academic performance.
